鹹香風味的調酒

Queen Mary

女王瑪麗
鹹味鮮味辛辣鹹香

材料

  • 伏特加50 毫升
  • 番茄汁120 毫升
  • 檸檬汁15 毫升
  • 伍斯特醬2 滴
  • 辣椒醬2 滴
  • 芹菜鹽1 撮
  • 黑胡椒1 撮

做法

在冰上調製；輕輕攪拌混合。

裝飾

芹菜莖，檸檬角，橄欖或辛辣泡菜

Bloody Mary

血腥瑪麗
鹹味鮮味辛辣柑橘草本鹹香

材料

  • 伏特加60 毫升
  • 番茄汁120 毫升
  • 檸檬汁15 毫升
  • 伍斯特醬2 滴
  • 辣椒醬2 滴
  • 辣根1 茶匙
  • 芹菜鹽1 撮
  • 黑胡椒1 撮

做法

將所有材料與冰塊放入搖酒器中。來回滾動以充分混合，或輕輕搖晃。濾入裝滿冰塊的高球杯中。根據喜好裝飾。

裝飾

芹菜莖、橄欖、檸檬角、萊姆角、培根條、醃製蔬菜（可選）

BLT Cocktail

BLT 雞尾酒
鹹味鮮味煙燻鹹香

材料

  • 伏特加45 毫升
  • 番茄汁90 毫升
  • 檸檬汁15 毫升
  • 伍斯特醬2 滴
  • 芹菜鹽1 撮
  • 黑胡椒2 撮

做法

將所有材料與冰塊搖勻，然後濾入裝滿冰塊的高球杯中。

裝飾

酥脆培根條，小番茄

Oyster Shooter

生蠔Shot
鹹味鮮味辛辣柑橘鹹香

材料

  • 伏特加30 毫升
  • 新鮮生蠔1 個
  • 檸檬汁15 毫升
  • 辣椒醬2 少許

做法

將所有成分混合在shot杯中。 立即飲用。

裝飾

可選：檸檬角或少許辣椒醬

Bloody Maria

血腥瑪麗亞
鹹味鮮味辛辣柑橘草本鹹香

材料

  • 龍舌蘭酒60 毫升
  • 番茄汁120 毫升
  • 萊姆汁15 毫升
  • 辣椒醬2 滴
  • 伍斯特醬2 滴
  • 芹菜鹽1 撮
  • 黑胡椒1 撮

做法

將所有材料放入裝有冰塊的搖酒器中。在搖酒器的兩半之間輕輕滾動以混合（或輕輕攪拌）。濾入裝滿冰塊的杯子中。根據需要裝飾。

裝飾

芹菜莖、萊姆角、橄欖和/或辛辣醃製蔬菜。

Red Snapper

紅鯛魚
鹹味鮮味柑橘草本辛辣鹹香

材料

  • 通用烈酒60 毫升
  • 番茄汁120 毫升
  • 檸檬汁15 毫升
  • 伍斯特醬2 滴
  • 辣椒醬2 滴
  • 芹菜鹽1 撮
  • 黑胡椒1 撮

做法

將所有材料放入裝有冰塊的搖酒器中。充分搖晃，然後倒入裝滿冰塊的高球杯中。

裝飾

芹菜莖、檸檬角、橄欖（可選）

Michelada

米切拉達
鹹味辛辣柑橘鮮味鹹香酵母

材料

  • 墨西哥拉格啤酒355 毫升
  • 檸檬汁30 毫升
  • 少許辣醬2 滴
  • 少許伍斯特醬2 滴
  • 一撮鹽1 撮
  • 一撮黑胡椒1 撮

做法

杯口沾鹽。加入檸檬汁、辣醬、伍斯特醬、鹽和胡椒。倒入墨西哥拉格啤酒。用檸檬角裝飾。

裝飾

檸檬角，鹽口

Gibson

吉布森
苦味草本鹹味鹹香

材料

  • 琴酒60 毫升
  • 不甜香艾酒10 毫升

做法

與冰塊攪拌後濾入冰鎮過的杯子。

裝飾

醃洋蔥

Bacon Bourbon Manhattan

培根波本曼哈頓
煙燻鹹味苦味木質鮮味鹹香

材料

  • 培根浸泡波本威士忌60 毫升
  • 甜味美思30 毫升
  • 安格式苦精2 滴

做法

與冰塊攪拌後過濾到冰鎮的玻璃杯中。

裝飾

酥脆的培根條

Dirty Martini

髒馬丁尼
鹹味苦味鮮味草本鹹香

材料

  • 琴酒60 毫升
  • 不甜苦艾酒20 毫升
  • 橄欖鹽水20 毫升

做法

將所有材料與冰塊攪拌均勻至冰鎮。 濾入冰鎮的馬丁尼杯中。

裝飾

橄欖（通常用串或雞尾酒籤串起 3 顆）

Caesar

凱撒
鹹味鮮味辛辣鹹香柑橘

材料

  • 伏特加45 毫升
  • 蛤蜊番茄汁120 毫升
  • 辣醬3 滴
  • 伍斯特醬4 滴
  • 檸檬角1 角
  • 鹽和胡椒2 撮

做法

在玻璃杯邊緣塗上芹菜鹽或香料混合物。將配料與冰塊混合在玻璃杯中，然後輕輕攪拌。大量裝飾。

裝飾

芹菜梗、檸檬角、醃豆、義大利香腸條、橄欖和香料杯緣。

血腥瑪格麗特

  • 檸檬汁15 毫升

  • 辣椒醬2 滴
  • 少許鹽1 撮
  • 少許黑胡椒1 撮

做法

將所有材料與冰塊搖勻，然後倒入裝滿冰塊的玻璃杯中。

裝飾

Chipotle Bloody Mary

墨西哥辣椒血腥瑪麗
辛辣鮮味鹹味煙燻鹹香

材料

  • 伏特加60 毫升
  • 番茄汁120 毫升
  • 檸檬汁15 毫升
  • 墨西哥辣椒在阿斗波醬中2 個
  • 伍斯特醬2 短劃線
  • 辣椒醬1 短劃線
  • 芹菜鹽1 撮
  • 黑胡椒1 撮

做法

將所有成分與冰塊混合在搖酒器中。 在搖酒罐之間來回滾動以混合。 濾入裝滿冰塊的玻璃杯中。 根據需要裝飾。

裝飾

芹菜梗、檸檬角、橄欖、醃製蔬菜、培根條、烤蝦或其他鹹味選擇。

Bull Shot

公牛特調
鹹味鮮味鹹香

材料

  • 伏特加45 毫升
  • 牛肉湯120 毫升
  • 伍斯特醬2 滴
  • 辣椒醬1 滴
  • 黑胡椒1 撮
  • 芹菜鹽1 撮

做法

調和法。將所有材料放入裝有冰塊的玻璃杯中。 輕輕攪拌。

裝飾

檸檬角或芹菜梗
